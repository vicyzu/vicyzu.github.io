Vision and Intelligent Computing (VIC) Lab Website

This repository hosts the official website of the
Vision and Intelligent Computing (VIC) Laboratory,
Department of Electrical Engineering, Yuan Ze University, Taiwan.

Live website: https://vicyzu.github.io


About VIC Lab

The Vision and Intelligent Computing (VIC) Lab conducts research in:

- Computer Vision
- Multimedia Processing
- Artificial Intelligence (AI)
- Intelligent and Interactive Computing

Our research emphasizes robustness, interpretability, and real-world deployability,
with applications spanning healthcare analytics, geoscience and environmental
monitoring, biosignal analysis, smart sensing, and autonomous systems.

The lab actively collaborates with international academic institutions and
industry partners, aiming to bridge fundamental research and real-world impact.


Website Contents

The website includes the following sections:

- About: Lab overview, mission, and announcements
- Publications: Automatically generated from BibTeX using Jekyll Scholar
- Projects: Ongoing and completed research projects
- Gallery: Lab activities, workshops, and academic events
- Team: Lab director, graduate students, undergraduate students, alumni, and interns


Technical Overview

- Static website built with Jekyll
- Hosted via GitHub Pages
- Publication management using Jekyll Scholar
- Content written in Markdown
- Customized layouts and styling for VIC Lab requirements

Local development is supported via Docker and Jekyll dev containers.
